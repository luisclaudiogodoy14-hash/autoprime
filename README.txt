
AutoPrime Veículos - Site estático (atividade)
=============================================

Arquivos incluídos:
- index.html (Home)
- estoque.html (Estoque)
- sobre.html (Sobre)
- servicos.html (Serviços)
- contato.html (Contato)
- styles.css (estilos)
- script.js (pequeno script para menu móvel)
- logo.svg (logo de exemplo)
- README.txt (este arquivo)

Instruções rápidas:
1. Substitua [NOME_1, NOME_2, NOME_3] no rodapé pelos nomes dos integrantes do grupo.
2. Para hospedar: envie a pasta para Netlify, Vercel ou GitHub Pages.
3. Se quiser formulários funcionais, conecte um backend ou use Netlify Forms / Formspree.
4. As imagens usam links de Unsplash; substitua por fotos reais se necessário.

Requisitos atendidos:
- Mínimo 5 páginas.
- Menu superior fixo com 5 links.
- Logo com link para home.
- Margens laterais com pelo menos 15% (desktop).
- Rodapé com nomes dos integrantes e UCB (substituir placeholders).
- Design responsivo e pronto para hospedagem.
